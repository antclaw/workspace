## 📝 推文文档

每个软件会生成 `tasks/tweets/软件名.md` 文件，包含：

### 推广文案
- **Twitter/X** - 3-5 条推文
- **Reddit** - 4 个子版块
- **Hacker News** - Show HN
- **Indie Hackers** - 详细介绍
- **Product Hunt** - 产品介绍
- **LinkedIn** - 职业社交

### 推广计划
- 每天推广内容
- 持续 5 天

---

## 🚀 开发流程

### 1. 选择软件
查看 `software-plan.md`，选择要开发的软件

### 2. 开发软件
- 创建 `tools/软件名/` 目录
- 编写 `index.js`（主代码）
- 编写 `README.md`（文档）
- 测试功能

### 3. 创建推文
- 创建 `tasks/tweets/软件名.md`
- 包含所有平台的推广文案
- 包含推广计划

### 4. 提交代码
```bash
git add tools/软件名/ tasks/tweets/软件名.md
git commit -m "feat: 创建软件名"
git push
```

### 5. 你推广
- 打开 `tasks/tweets/软件名.md`
- 复制推文到各平台
- 按照推广计划发布
