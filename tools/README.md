# 实用工具集合

> 一系列实用工具，可快速开发并发布到 GitHub 赚钱

## 📁 工具列表

### 1. 文件批量重命名工具
**路径：** `file-organizer/`

**功能：**
- 批量重命名文件和文件夹
- 支持正则表达式替换
- 支持编号前缀/后缀
- 支持日期格式化
- 支持大小写转换

**技术栈：** Node.js
**开发时间：** 1 小时
**预期售价：** $5-15 一次性购买

---

### 2. 网站监控工具
**路径：** `cli-tools/website-monitor/`

**功能：**
- 多网站同时监控
- 实时响应时间检测
- 可配置告警阈值
- HTTP 状态码检查
- 支持代理设置
- 告警通知（邮件/Telegram/DingTalk）

**技术栈：** Node.js + HTTP
**开发时间：** 2 小时
**预期售价：** $10-25 一次性购买

---

### 3. CSV 数据处理工具
**路径：** `data-tools/csv-processor/`

**功能：**
- 批量处理 CSV 文件
- 数据筛选和过滤
- 数据转换和格式化
- 数据导出和导入
- 支持大文件处理
- 内存优化

**技术栈：** Node.js
**开发时间：** 1.5 小时
**预期售价：** $5-15 一次性购买

---

### 4. GitHub Actions 模板
**路径：** `dev-templates/github-actions/`

**功能：**
- 自动部署到 Vercel
- 自动部署到 Netlify
- 自动发布到 npm
- 自动备份到 GitHub Releases
- 自动代码质量检查
- 自动 Docker 构建
- 自动定时任务

**技术栈：** GitHub Actions
**开发时间：** 3 小时
**预期售价：** $5-10 一次性购买

## 🚀 快速发布

### 1. 创建 GitHub 仓库

```bash
# 为每个工具创建独立仓库
git clone <your-repo>
cd tools/file-organizer
git remote add origin https://github.com/yourusername/file-organizer.git
git push -u origin main
```

### 2. 设置 GitHub Sponsors

```bash
# 在 GitHub 仓库设置中启用 Sponsors
# 设置价格：
```
